## 简介
你独自一人来到一处陌生领地，四周寂静无人。  
据传言，该地居住着怪异之物，随时准备袭击冒然来访者。  
你能够顺利逃出这是非之地吗？  
也许你会被人发现遗物……  
当然，你可以选择拯救队友，一起逃出生天。  
但这无疑是火上浇油。  
你会接受这份独一无二的挑战吗？  

## 玩法
在地图中每个队伍随机出生到一个陌生的地方，在有限的地图内与其他玩家对战，活到最后为最后的赢家。
